# Code

        This public repository is organized around a demo workflow. The demo scripts use synthetic inputs with the same column names and units as the restricted model inputs. The full analysis can be run by replacing the demo paths in `config/demo_config.yml` with authorized local Strava and SDGSAT-1 data.

        The public code avoids obsolete intermediate outcomes and uses the final manuscript convention: female nighttime PA volume is the primary response, male PA is a reference response, and city-level female-to-male PA ratio change is a post-hoc policy diagnostic.
